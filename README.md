Этот веб-сервис принимает адрес от пользователя, выполняет его стандартизацию через API DaData и возвращает стандартизированный адрес.


## Пример ответа

```json
{
    "country": "Россия",
    "city": "г Москва",
    "street": "ул Сухонская",
    "houseNumber": "11",
    "flatNumber": "89"
}
